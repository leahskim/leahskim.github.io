Personal Website

This website is my first personal project I’ve ever published online. As I was writing up my resume, I realized that I’ve so far only learned from workshops and tutorials, but haven’t applied my knowledge to create something of my own. I figured that creating a website would be a perfect way to test my skills as well as introduce myself to others in a creative way. I used the Bootstrap framework, HTML, and CSS to create this site.

I started out with a rough sketch of what I wanted each page to look like, in terms of font size, colour, and layout. I was initially planning on making separate pages for each section, as shown in the sketches, but later realized how short each of my pages would actually be if made individually. Instead, I merged them all into one long home page with the navigation links anchored to each section. I also ended up taking out the whole experience page because I felt that it made the website would look less like a portfolio and more a resume.

Although it may say 2020 for the creation date, my personal website has no real due date because I will constantly be learning new skills in the future that will help me improve it. This specific website is only the first step in my journey to become an innovative designer and an efficient problem-solver.
