Personal Website

This website is the first personal project I have ever published online. As I was writing up my resume, I realized that I’ve so far only learned from workshops and tutorials, but haven’t applied my knowledge to create something of my own. I figured that creating a website would be a perfect way to test my skills as well as introduce myself to others in a creative way. I used the Bootstrap framework, HTML, and CSS to create this site.

Experience Page Sketch

Projects Page Sketch

I started with a rough sketch of what I wanted each page to look like, in terms of font size, colour, and layout. I was initially planning on making separate pages for each section, as shown in the sketches, but later realized how short each of my pages would be if made individually. Instead, I merged them all into one long home page with the navigation links anchored to each section. I also ended up taking out the whole experience page because I felt that it made the website would look less like a portfolio and more like a resume.

As I constantly learn new skills in the future, I will continuously improve this website. This is only the first step in my journey to becoming an innovative designer and an efficient problem-solver.
